# متجر وسيل ماركت الإلكتروني

## وصف المشروع
وسيل ماركت - متجر إلكتروني عربي متكامل لبيع المنتجات المادية المختلفة. 
منصة تسوق عربية توفر أفضل المنتجات في جميع الفئات مع خدمة توصيل سريعة.

## الميزات
- ✅ عرض المنتجات بفئات متنوعة
- ✅ بحث وتصفية متقدمة
- ✅ سلة تسوق متكاملة
- ✅ لوحة تحكم للمسؤول
- ✅ نظام إدارة المخزون
- ✅ عروض وتخفيضات
- ✅ تقييمات المنتجات
- ✅ دعم كامل للغة العربية
- ✅ تحسين محركات البحث (SEO)

## التقنيات المستخدمة
- Firebase Firestore
- Firebase Authentication  
- Firebase Storage
- JavaScript (ES6+)
- HTML5 / CSS3
- Responsive Design

## فئات المنتجات
- إلكترونيات
- موضة
- المنزل
- الجمال
- رياضة
- كتب
- طعام
- صحة

## كيفية التشغيل
1. افتح `index.html` في المتصفح
2. أو انتقل إلى: https://wacelmarkt.netlify.app
3. البدء في تصفح وتسوق المنتجات

## لوحة التحكم
للوصول إلى لوحة التحكم:
1. سجل الدخول باستخدام:
   - البريد: admin@wacelmarkt.com
   - كلمة المرور: Admin123456
2. انقر على زر "لوحة التحكم" في الشريط العلوي
